# 计算机原理实验
## `experiment_*`
> 所有工程使用`make`构建，Stm32创建工程是使用了`copy all`mode，因此结果基本可以复现

是 `April` 编写的四次推展实验及其报告，其对应的任务是
1. 花样流水灯

  本人在代码中实现了
  - 数码管+交通灯+LED的流水灯
  - 两个按钮调节闪烁速度
2. FPGA双端口RAM储存实验
  
  由于桥接板实际上是将LCD、ARM、FPGA三者连接在一起，很容易发生意料之外的时序错误，因此使用排线，故而调整任务指标是每个RAM两位地址、每个地址两位数据、两位输出、一个写使能信号
  
  我于是实现了
  - 按钮0控制写入
  - 按钮1控制运算
  - sw0控制写使能
  - sw1控制片选
  - sw2、sw3控制待写入数据
  - sw4、sw5控制写入/读取地址
  - 左侧4个数码管显示当前数据
  - 右侧两个数码管显示待写入数据
  - LED状态灯
3. 中断及抢占
  
  我实现了
  - 数码管按秒跳变
  - 低级别的交通灯中断(可以打断数码管计数)
  - 高级别的LED流水灯中断(可以打断上面两个)
4. 自拟实验

  我实现了
  - LCD点亮+显示
  - FPGA模型部署
  - 串口通信
